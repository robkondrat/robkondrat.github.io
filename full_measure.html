Actualize Group Project - Watch out for dog poop!

We tried to have fun with this group project while still implementing some of the concepts we learned in class as well as additional tools.

My partner and I chose this so we could learn how to utilize Google's Maps Javascript API to display a map based on an address submitted by a user.

We also wanted to try using Active Storage for the first time to allow users to upload images (we chose a cartoon example to spare the audience).

Future releases would not only include an interactive map but give the user an option to set a walking route with the least amount of reported poops.

Tools

The app is built with a Ruby on Rails backend and Vue.js frontend, which were part of the curriculum in the bootcamp.

I researched and implemented technology that was not covered in bootcamp, such as Action Cable to update live data on a page without the user refreshing their browser, CronJob to change expired data, and ActiveStorage to handle files uploaded by users.
